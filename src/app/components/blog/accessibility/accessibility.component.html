Accessibility

  1. Semantic HTML

  2. Content importat img should have alt attribute

  3. Create logical structure (header > main > nav > h1 > h2 > section > footer, etc)

  4. Add aria attributes to interactive element, that does not have a text

  5. Make your site keyboard friendly

  6. Make sure your site meets color contrast requirements

  7. Read Web Content Accessibility Guidelines (WCAG)

  8. Use Angular features (title, a11y sdk)

  9. Test your site with keyboard, lighthouse, screen reader (NVDA, JAWS, Voice Over), etc
